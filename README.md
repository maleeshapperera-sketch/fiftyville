# fiftyville
Fiftyville - CS50 Mystery
A SQL detective project where I solved a theft case by analyzing a town database.

🕵️ The Case
The CS50 Duck was stolen from Fiftyville on July 28, 2025. Using SQL queries, I identified:

The thief - who stole the duck

The escape city - where they fled

The accomplice - who helped them escape

🛠️ What I Used
SQLite

Complex queries with JOINs, subqueries, and aggregations

Crime scene reports, interviews, phone logs, ATM records, flight data

📁 Files
fiftyville.db - The investigation database

log.sql - All SQL queries with comments documenting my investigation process

answers.txt - Final case solution
